### Understanding Plot Generation

When matplotlib creates a plot without `plt.show()`, it:
1. **Processes the data**: Converts our lists into plot coordinates
2. **Creates the figure**: Generates the plot in memory
3. **Saves automatically**: May save to a file (depending on backend)
4. **Doesn't display**: No visual output without explicit show command

## üö® Common Mistakes and Troubleshooting

### 1. Missing matplotlib Installation

```python title="Error - Module not found"
import matplotlib.pyplot as plt  # ModuleNotFoundError
```

```bash title="Solution - Install matplotlib"
pip install matplotlib
```

### 2. Forgetting to Display Plot

```python title="Problem - No visual output"
plt.plot(x, y)
# Missing plt.show() - plot created but not displayed
```

```python title="Solution - Always add show()"
plt.plot(x, y)
plt.show()  # Essential for displaying the plot
```

### 3. Data Length Mismatch

```python title="Problem - Different length arrays"
x = [1, 2, 3]
y = [4, 5, 6, 7, 8]  # Different lengths cause errors
plt.plot(x, y)
```

```python title="Solution - Match array lengths"
x = [1, 2, 3, 4, 5]
y = [4, 5, 6, 7, 8]  # Same length
plt.plot(x, y)
```

### 4. Missing Import Statements

```python title="Problem - Using numpy without import"
data = np.random.randn(100)  # NameError: name 'np' is not defined
```

```python title="Solution - Import required libraries"
import numpy as np
data = np.random.randn(100)
```
